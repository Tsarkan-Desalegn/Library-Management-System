# 📚 Library Management System

## 🔹 Overview
This project is a simple **Library Management System** built with **HTML, CSS, and JavaScript**.  
It provides a **sidebar navigation menu** and a **main content area** where users can add and display book details.



## 🔹 Features

### 🗂 Sidebar Navigation
- Toggle menu with open/close icons (hamburger & cancel)  
- Links for **Home** and **Calendar** sections  

### 🏠 Main Content (Home Section)
- Input fields for book details:  
  - Title  
  - Author  
  - Shelf number  
  - Date  
- Button to **add a book**  
- Display section to show entered book information  

### 📱 Responsive Design
- Sidebar can be toggled using the checkbox + SVG icons  
- Layout adapts to different screen sizes
