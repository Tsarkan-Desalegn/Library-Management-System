# 📚 Library Management System

A simple and interactive **Library Management System** built with **HTML, CSS, and JavaScript**.  
This project allows users to add, borrow/return, and delete books, while displaying them in a clean, styled interface.

---

## 🚀 Features
- ➕ Add books with **title, author, shelf number, and registration date**  
- 📖 Borrow/Return toggle — switch book status between *Available* and *Borrowed*  
- 🗑️ Delete books — remove books from the library collection  
- 🎨 Styled interface with sidebar navigation and responsive design  
- 📅 Human‑friendly date formatting for registration  

---

## 🛠️ Technologies Used
- **HTML5** → structure and layout  
- **CSS3** → styling and sidebar design  
- **JavaScript (ES6)** → dynamic book management

## ⚙️ How It Works
1. Enter book details (title, author, shelf number, date).  
2. Click **Add Book** → book card appears in the display section.  
3. Use **Borrow/Return button** → toggles book availability.  
4. Use **Delete button** → removes the book card completely.
